1. Setup environement to run

1) Setup packages
	- Install PyEnchant package in terminal
		pip install pyenchant
	- Install tweepy-3.5.0
	
2) Setup database
	- Setup mysql
	- Load data in 'database.sql'
	
2. Run the project.

In terminal, run the following command.

$ python manage.py runserver 127.0.0.1:8000

In web browser, go to http://127.0.0.1:8000/login
